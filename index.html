THE MENTAL MODEL

SQL isn’t executed
in the order you write it.

You write SELECT first, but the database begins at FROM. It builds a working set, transforms it clause by clause, and only then returns the result.

WRITTENSELECT → FROM → WHERE → GROUP BY → ORDER BYEXECUTEDFROM → WHERE → GROUP BY → SELECT → ORDER BY
